All examples live in the `examples/` directory and can be run immediately.

## Running an example

```bash
# Clone the repo
git clone https://github.com/jacobcwright/open-animate.git
cd open-animate

# Install workspace dependencies
pnpm install

# Run any example
cd examples/hello-world
npx remotion studio   # preview in browser
npx oanim render      # export to MP4
```

## hello-world

The simplest possible oanim project â€” a single `fadeUp` animation on centered text. Start here to understand the basics.

**Uses:** `fadeUp`, `Background`, `SafeArea`, `palettes`

## launch-video

A 4-scene product launch video with transitions between scenes. Demonstrates TransitionSeries with `fadeBlur` and `scaleFade` transitions, plus staggered element entrances.

**Uses:** `TransitionSeries`, `fadeBlur`, `scaleFade`, `fadeUp`, `popIn`, `AnimatedCharacters`, `Background`, `SafeArea`, `Badge`, `Card`

## logo-reveal

Animated logo entrance with a glow effect. Shows how to combine `popIn` with `GlowOrb` for a polished reveal.

**Uses:** `popIn`, `blurIn`, `GlowOrb`, `Background`, `Vignette`

## meme-caption

Vertical-format (1080x1920) social media clip with bold captions. Demonstrates non-standard resolutions via `animate.json`.

**Uses:** `fadeUp`, `elasticScale`, `Background`, `SafeArea`

## explainer

4-scene step-by-step explainer with numbered sections. Uses `AnimatedCharacters` for headers and staggered `fadeUp` for bullet points.

**Uses:** `TransitionSeries`, `wipe`, `clipCircle`, `AnimatedCharacters`, `fadeUp`, `Background`, `SafeArea`, `Grid`

## investor-update

Metrics dashboard with animated `CountUp` numbers. Demonstrates data-driven animations with prefix/suffix formatting.

**Uses:** `CountUp`, `fadeUp`, `Card`, `Background`, `SafeArea`, `palettes.midnight`

# Open Animate

**The open-source creative suite for AI agents.**

Give Claude Code, Codex, and other coding agents the ability to generate images, compose videos, and render motion graphics — all through code.

```bash
npx oanim init my-video
```

## Capabilities

<CardGroup cols={2}>
  <Card title="Asset Generation" icon="images">
    Generate images, edit photos, remove backgrounds, upscale — multi-provider AI media gateway with fal.ai, Runway, and more.
  </Card>
  <Card title="Video Composition" icon="film">
    Compose React-based videos with Remotion. 8 animation presets, 14 transitions, typography, and production-ready UI components.
  </Card>
  <Card title="Design System" icon="palette">
    5 color palettes, font stacks, spacing scale, and components like Terminal, Card, Badge — all tuned for 1920x1080.
  </Card>
  <Card title="Cloud Rendering" icon="cloud">
    Render locally or ship to the cloud. No Chromium or ffmpeg install needed for `oanim render --cloud`.
  </Card>
</CardGroup>

## How it works

```
oanim init        → scaffold a Remotion project with @oanim/core
oanim render      → export MP4 (local or --cloud)
oanim assets      → generate and manipulate images with AI
```

## Packages

| Package | Description |
|---------|-------------|
| `oanim` | CLI — project scaffolding, rendering, asset generation |
| `@oanim/core` | Animation presets, transitions, components, design tokens for Remotion |

## Platform

Open Animate is **open-core** — everything is open source (Apache 2.0) and self-hostable. The hosted platform at `api.oanim.dev` handles auth, billing, cloud rendering, and the media gateway so you don't have to.

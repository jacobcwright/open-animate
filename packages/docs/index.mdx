# The creative suite for agents.

Open-source toolkit that turns any coding agent into a motion designer. Describe what you want. Get an MP4.

```bash
npx oanim init my-video
```

Your agent writes React. [Remotion](https://remotion.dev) renders frames. You get an MP4.

## Works with

Claude Code · Cursor · Codex · Windsurf · any agent that writes code
